# MA-RLHF 课程索引

> **来源**：[MA-RLHF](https://github.com/dhcode-cpp/MA-RLHF)（dhcode-cpp）
> **定位**：LLM 算法工程师从零到可面试的完整工程课程。10 个专题 + xtrain 专题，从 Transformer 手写到 RLHF 全链路，从单卡到分布式，从推理框架到 R1 复现。
> **笔记位置**：笔记分布在 `AI/3-LLM/` 各子目录，本文是统一入口。

---

## 课程结构

```
MA-RLHF 课程
├── lc1   基础组件          ← Tokenizer / Embedding / 位置编码
├── lc2   Transformer       ← 完整 Encoder-Decoder 手写
├── lc3   GPT 系列          ← Decoder-only / KV Cache / BPE / Muon
├── lc4   Llama 系列        ← RoPE / GQA / SwiGLU / RMSNorm / TPA
├── lc5   DeepSeek V3       ← MoE / MLA / MTP / YaRN / mHC / Muon
├── lc6   SFT 全链路        ← 数据 / LoRA / 完整训练 / Continue Pretraining
├── lc7   RL 基础           ← REINFORCE / DQN / 策略梯度 / GAE
├── lc8   RL×LLM            ← RLHF-PPO / Bradley-Terry / DPO / IPO / KTO / GRPO / PRM / O1
├── lc9   分布式 RL 训练    ← Ray 三角架构 / 异步 GRPO / verl 实战 / R1 复现
├── lc10  推理系统          ← Continue Batching / PageAttention / vLLM V0→V1 / PD分离 / SD
└── xtrain 分布式并行手写   ← DP / ZeRO / TP / PP(DualPipe) / CP(RingAttn) / EP(MoE)
```

---

## 推荐学习路径

**路径 A：架构面试速通（1.5 天）**
lc1 → lc2 → lc3 → lc4 → lc5
重点：Transformer 手写 → GPT/Llama 演进 → DeepSeek MLA/MoE

**路径 B：RL 对齐面试速通（1.5 天）**
lc7 → lc6 → lc8 → lc9
重点：RLHF 四模型架构 → GRPO vs PPO → R1/verl 实战

**路径 C：推理系统面试速通（1 天）**
lc10 顺序过：Continue Batching → PageKVCache → PageAttention → vLLM V0→V1 → Chunked Prefill → PD 分离

**路径 D：分布式训练硬核（2 天）**
xtrain-01 → 02 → 03 → 04 → 05 → 06 → 07
重点：从零手写，不依赖 DeepSpeed/Megatron

**路径 E：全栈 LLM 工程师（5 天）**
路径A → 路径B → 路径C → 路径D
