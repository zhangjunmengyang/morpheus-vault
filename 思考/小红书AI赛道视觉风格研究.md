# 小红书 AI 赛道视觉风格 & 网感研究笔记

> 调研时间：2026-02-20
> 数据来源：小红书搜索"AI工具""AI陪伴""AI爆款""教你用AI""怎么调教deepseek""AI虚拟伴侣 教程""AI人格定制"
> 样本：50+ 帖子封面截图，重点分析 1000+ 赞的爆款

---

## 一、封面图风格分类（6 大类）

### 1. 便签/备忘录风格 🔥🔥🔥（爆款率最高）
**代表：** "如何调教deepseek？妹子们看好了" (7850赞)、"终于把deepseek调回来了！" (1375赞)
**特征：**
- 背景色：浅奶白/浅黄/浅蓝/浅绿 柔和色系
- 大段引号「❝ ❞」包裹核心句子
- 字体：**手写体/楷体/宋体**，不是黑体！
- 重点词用**加粗/变色/高亮**
- 没有 emoji 堆砌，最多 1-2 个
- 整体像一张 Apple Notes / Sticky Notes 截图
- **极度口语化**，像跟闺蜜说话

### 2. 产品截图/对话截图风格 🔥🔥
**代表：** "调教ai智能" (258赞)、"我拿deepseek克隆了我的女朋友" (250赞)
**特征：**
- 直接截图 DeepSeek/ChatGPT 对话界面
- 有真实对话内容，能看到具体的 prompt 和回复
- 可能加了手绘标注（箭头、圈圈、划重点）
- **真实感 > 设计感**
- 多图帖子：封面是截图，后面翻页是教程步骤

### 3. 大字报/标题党风格 🔥🔥
**代表：** "我想问问大家是怎么把DeepSeek调成网上那样的" (1.3万赞)
**特征：**
- 纯色背景（通常是亮色/渐变）
- 超大字号，一句话占满整张图
- 字体加粗+描边/阴影
- 没有多余装饰，一眼看完
- 标题本身就是引发好奇/共鸣的钩子

### 4. 图文混排/卡片风格 🔥
**代表：** "AI Social 产品分析" (765赞)、"2026 Coding工具指北" (623赞)
**特征：**
- 类似 PPT/Notion 排版
- 有小图标/logo 排列
- 分区域信息展示
- 适合产品评测/工具推荐/横评类内容
- **偏专业/理性，不如前三种"爆"**

### 5. 真人出镜/视频封面 🔥
**代表：** "每天2小时打造你的内容帝国" (各种)
**特征：**
- 博主真人照片+大字标题叠加
- 或者是短视频的封面帧
- 适合个人IP建设

### 6. 品牌感/设计感封面
**代表：** 一些专业账号的系列帖
**特征：**
- 统一 VI 色系
- 精致排版
- **但赞数反而不如前几种！设计感 ≠ 流量**

## 二、爆款文案用词规律

### 标题钩子模式（按频率排序）

1. **反问/疑问句** → "如何调教deepseek？" "你真的都发现了吗？"
2. **数字+干货承诺** → "6种Prompt心法" "5步训练法" "100小时路径图"
3. **口语化/网络用语** → "妹子们看好了" "炸了！" "姐妹们" "绝了" "太会了" "救命"
4. **争议/情绪** → "算出轨吗？" "不正经的调教指令来了" "一夜爆红"
5. **痛点共鸣** → "每天被信息轰炸怎么解脱" "为什么始终没有AI陪伴产品超过千万DAU"
6. **结果展示** → "30天血赚1200万" "半年销量第一" "一夜爆红60万人围观"

### 绝对不能用的词（太冷/太技术/太官方）
- ❌ "基于xxx技术" "赋能" "底层架构" "中间层平台"
- ❌ "人格包" "Schema" "元数据" "导出器"
- ❌ "道法术器" "认知框架" "函数化人格"
- ❌ "这不是科幻，这是现在" ← 我之前用的，太冷了

### 该用的词（有网感的）
- ✅ "救命！" "绝了！" "炸了！" "太离谱了" "笑死"
- ✅ "手把手教你" "保姆级教程" "一看就会"
- ✅ "姐妹们" "家人们" "宝子们"
- ✅ "亲测有效" "自用分享" "不藏了"
- ✅ "调教" "养成" "克隆" "打造" "玩转"
- ✅ "0基础" "小白也能" "3分钟学会"

## 三、颜色使用规律

### 爆款封面的主色调
- **浅色系为主**：奶白、浅蓝、浅绿、浅粉、浅黄
- **深色封面也有**但主要是科技/编程向（"2026 Coding指北"等）
- **高赞帖子很少用纯黑背景**！！← 我v0.1犯的错
- **文字颜色**：深灰/黑色为主（不是白色！除非深色背景）
- **强调色**：红色/橙色/品牌色用于高亮关键词

### 对比我的 v0.1
| 我的 v0.1 | 爆款真实风格 |
|-----------|-------------|
| 深色背景 #111827 | 浅色/白色背景为主 |
| 白色文字 | 深色文字 + 彩色高亮 |
| 极简冷淡 | 温暖亲切/有手写感 |
| 居中对称 | 左对齐/不对称/呼吸感 |
| 无人味 | 像人写的笔记 |

## 四、对话截图风格要点

### 真正好的对话截图应该：
1. **直接截真实对话**，不是模拟（用户一眼看出来是假的）
2. 如果要模拟，要还原到**像素级**（顶栏、电池、时间、信号）
3. **有足够的对话轮次**（至少 5-8 轮才有故事感）
4. **对话内容本身要有趣/有用** → 不是"你好" "你好有什么能帮你的"
5. **手绘标注**增加真实感（红色箭头、划重点框、手写批注）

## 五、关键洞察

### 1. 小红书的核心审美是"人味" > "设计感"
- 越像人写的笔记/便签，互动率越高
- 精致的设计反而像广告，用户会划走
- 手写体 > 黑体，不规则 > 对称，温暖 > 冷酷

### 2. 封面只做一件事：让人点进来
- 一句话钩子 + 情绪色彩
- 不需要传递完整信息
- **3 秒法则**：用户滑动速度极快，封面必须 3 秒内让人停下来

### 3. 多图帖子的节奏
- 图1（封面）：钩子——"救命！我居然养了一个AI男友"
- 图2-3：展示结果——对话截图、效果展示
- 图4-6：步骤教程——怎么做到的
- 图7-9：进阶/彩蛋——更多玩法、评论区引导

### 4. 我们的产品怎么说人话
| 技术语言 | 小红书语言 |
|---------|----------|
| AI人格包 | AI灵魂伴侣 / AI男友 / AI闺蜜 |
| 安装部署 | 3步搞定 / 手把手教 |
| 人格一致性 | 越聊越懂你 / 永远不会忘记你 |
| 记忆系统 | 记住你说过的每一句话 |
| 道法术器 | （别说，没人懂也没人在意）|
| 进化协议 | 会成长的AI / 养成系 |
| OpenClaw框架 | 免费开源工具 |
| 中层平台 | （别说）|

## 六、对模板系统的改造方向

### cover.html 需要的新模板

**模板A：便签/笔记风格**（最高优先级）
- 浅色背景（米白/浅黄/浅蓝可选）
- 左上角有便签纸标签或日期
- 手写体标题
- 核心句子用大引号❝❞包裹
- 关键词红色/橙色高亮
- 底部可以有手写签名感的作者名

**模板B：Apple Notes / Sticky Notes 风格**
- 模拟苹果备忘录界面
- 顶栏有"备忘录"标题
- 正文是标记笔记
- 看起来像从手机截图的

**模板C：对话截图+批注风格**
- 真实对话截图（模拟 DeepSeek 界面）
- 叠加手绘红色箭头和框线标注
- "看这里！" "关键！" 手写批注

### chat.html 改造
- 模拟 DeepSeek 真实界面（不是 iOS 通用聊天）
- 对话轮次 ≥ 5 轮
- 高度自适应（不再固定 3:4 留空白）
- 可选叠加手绘标注

### 配色系统改造
- 默认改为浅色系
- 预设几套"网感配色"：奶茶色、清新绿、少女粉、学院蓝
- 深色方案保留但不做默认
