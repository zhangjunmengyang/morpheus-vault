# 海量数据去重

### **海量数据高效去重**

- RocksDB 状态后端 + mapstate(K-V)
- Hyperloglog：一种统计估计方法，基数计数，不 100%准确
- 布隆过滤器：说没有一定没有，说有就可能有，不 100%准确
- BitMap 位图：每个 bit 位标记某个元素对应的 value
- 估算 20 亿 个 int（4 字节，32bit） 空间大约是 8 G
- 如果改用 bitmap（ 一个 bit 存一个），0.25 G
- 如果对象不是数字，可以用表中 ID，或者映射一下
- 外部数据库：比如 Redis
