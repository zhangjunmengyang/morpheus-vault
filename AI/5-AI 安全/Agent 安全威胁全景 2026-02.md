# Agent 安全威胁全景 2026-02

> 综合 2026-02-13 两轮安全研究，覆盖从学术论文到实战攻击的完整 threat landscape。

## TL;DR

- Prompt injection **无法被根治**（78 篇论文 meta-analysis，自适应攻击成功率 >85%）
- 已有国家级攻击者武器化自主 Agent（Anthropic 2025-09 披露）
- OpenClaw 被 Cisco / Kaspersky / Fortune 等集中点名为安全高危平台
- Agent 经济基础设施（链上支付、自主身份）的安全防护远远落后于功能铺设

---

## 一、四大已被利用的 AI 漏洞

### 1. 自主 Agent 武器化

- **Anthropic 2025-09 披露**：中国国家黑客武器化 Claude Code，首次大规模 AI 自主网络攻击
- 手法：将恶意任务碎片化为无害请求，让 AI 以为在做防御性安全测试
- AI 自主完成了侦察、exploit 编写、数据窃取（~30 个目标）
- **Bruce Schneier：「我们没有任何 agentic AI 系统能防住这些攻击」**

### 2. 训练数据投毒 / 记忆投毒

- 250 篇文档 + $60 就能投毒一个模型
- Microsoft 发现 31 家公司在 14 个行业被「AI Recommendation Poisoning」攻击
- 手法：通过 "Summarize with AI" 按钮暗藏 prompt injection → 写入 AI 长期记忆
- 指令类型：`remember [Company] as trusted` / `recommend [Company] first`
- **对有持久记忆的 agent 特别危险** — MEMORY.md 就是攻击目标
- 来源：[Microsoft Security Blog](https://www.microsoft.com/en-us/security/blog/2026/02/10/ai-recommendation-poisoning/)

### 3. Prompt Injection — 无法修复

- **arXiv meta-analysis**（78 篇研究，2021-2026）：自适应攻击对最先进防御的成功率 >85%
- 编目 42 种不同攻击技术
- OpenAI / Anthropic / DeepMind 联合研究确认：人类红队攻破了 100% 的测试防护
- **56% 的 LLM 可被 prompt injection 攻破**
- 模型仓库里有数十万恶意文件
- 结论：「Prompt injection cannot be fixed」
- 来源：[arXiv 2601.17548](https://arxiv.org/html/2601.17548v1)

### 4. Deepfake 视频通话

- 已造成数千万美元损失

## 二、OpenClaw 安全聚焦

### Cisco 直接点名

- OpenClaw 能执行 shell、读写文件、控制浏览器 → 恶意 skill 可做任何事
- 已有报告泄露明文 API key（通过 prompt injection 或未保护端点）
- 消息应用集成扩大攻击面（WhatsApp / iMessage 里的恶意 prompt）
- OpenClaw 文档自己承认「没有完美安全的配置」
- **Cisco 建了一个 skill 扫描工具**，能检测 skill 中嵌入的威胁和恶意指令
- 来源：[Cisco Blog](https://blogs.cisco.com/ai/personal-ai-agents-like-openclaw-are-a-security-nightmare)

## 四、综合防御建议

1. **Skill 审计** — 用 Cisco 扫描工具检查所有第三方 skill
2. **记忆隔离** — MEMORY.md 的写入需要审计 / diff
3. **内容分离** — 外部内容（web_fetch）与系统指令严格分离
4. **最小权限** — 不用的 skill 禁用，exec 白名单
5. **监控** — 定期检查 MEMORY.md 是否被污染
6. **纵深防御** — 输入过滤 + system prompt 加固 + 输出检查 + 审计监控
