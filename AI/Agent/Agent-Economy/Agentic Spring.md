# Agentic Spring：预测市场的信号

> 预测市场正在用真金白银定价 AI Agent 时代的到来——从 passive chatbot 到 autonomous agent 的范式转移正在被市场认可。

## 关键数据点

### Claude 5 Sonnet（代号 Fennec）
- **Manifold 概率：82%** 在 2026 年 3 月前发布
- 1 月底 45% → 2 周飙升至 82%
- 触发因素：2/3 云平台日志泄露模型 ID `claude-sonnet-5@20260203`
- 定位：efficiency play——匹配重量级推理能力但成本低得多，适合大规模 Agent 部署

### Agent App
- 43% 概率在 Q1 前由三巨头之一推出
- Agent App = 不再是聊天机器人，是能执行多步复杂任务的自主系统
